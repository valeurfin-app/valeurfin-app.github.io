📜 Privacy Policy

Effective: April 2026 | v4.0

1. Data Collection

ValeurFin collects ZERO data on any external server. All information is stored exclusively on your device using IndexedDB, localStorage, sessionStorage, and cookies as fallback.

2. Storage

Data is stored locally with quad-redundancy. No tracking, no third-party sharing.

3. Third-Party

MFAPI for fund search, Yahoo Finance for live prices (only stock symbols sent), TradingView widgets for market visualization. Optional: Gemini AI if user provides their own API key.

4. Deletion

Security → Wipe All Data to permanently delete.

5. Contact
