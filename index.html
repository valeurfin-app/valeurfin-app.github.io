📜 Privacy Policy

Effective: April 2026 | Product: ValeurFin v3.0

1. Data Collection

ValeurFin collects ZERO personal data on any external server. All information you enter — financial data, names, PINs, and preferences — is stored exclusively on your device using browser IndexedDB and localStorage.

2. Data Storage

All data is encrypted using AES-256 encryption and stored locally. No cookies, no tracking pixels, no analytics services, no third-party data sharing.

3. Third-Party Services

ValeurFin uses the MFAPI (api.mfapi.in) solely for mutual fund scheme name search. No personal or financial data is transmitted.

4. Data Deletion

You can permanently delete all stored data at any time via Security → Wipe All Data.

5. Contact

For questions: valeurfin.masteryourfinances@gmail.com
